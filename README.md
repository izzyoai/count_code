# count_code( 代码计数)

## 1.功能

- 统计.cpp .h .c .java .js文件的代码
- // /**/ 暂时只识别这两种注释
- 统计，并给出注释
- - 文件大小
  - 文件总行数
  - 文件代码行数
  - 文件空白行数

- 利用可视化表格

- | 文件名 | 类型 | 大小 | 总行数 | 代码行数 | 注释行数 | 空白行数 | 路径 |
  | ------ | ---- | ---- | ------ | -------- | -------- | -------- | ---- |
  |        |      |      |        |          |          |          |      |

- 可以选择文件或递归文件夹下的匹配文件(.cpp .h .c .java .js)
